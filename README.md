# 🤖 AI FAQ Chatbot

An AI-powered FAQ chatbot built with **Python, Flask, and NLP** that answers user queries using **TF-IDF Vectorization** and **Cosine Similarity**. It features a modern ChatGPT-inspired interface, voice input, speech output, and typing animation for an interactive user experience.

## 🚀 Features

- AI-powered FAQ responses
- ChatGPT-style UI
- Voice Input (Speech-to-Text)
- Voice Output (Text-to-Speech)
- Typing Animation
- Responsive Design

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Pandas
- Scikit-learn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python3 app.py
```

Open: `http://127.0.0.1:5000`
